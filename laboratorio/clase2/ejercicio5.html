Codifique en el lenguaje JavaScript la solución al siguiente problema:
 
		Ingresar mediante una ventana de diálogo, valores numéricos.
		Por cada valor ingresado, se debe informar inmediatamente, mediante una alerta, el doble de dicho valor.
		 
		Se deberán ingresar valores una y otra vez hasta que se ingrese un CERO, el cual indicará el fin del programa.
		NOTA 1: Si se ingresa un CERO como el primer valor, el programa se detendrá inmediatamente.
		NOTA 2: Este ejercicio NO requiere la validación de los datos de entrada, asuma que serán siempre números.
