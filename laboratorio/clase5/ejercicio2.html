Programando en JavaScript, cree un formulario que contenga:

		Un conjunto de radio buttons, que permitan seleccionar el estado civil de una persona.
		Un botón.

		Al presionar el botón, deberá informarse mediante una ventana de alerta, el valor seleccionado; 

		y posteriormente, enviar el formulario.

		En caso de no haber seleccionado ninguna opción, informar dicha situación con una leyenda de error, 
		también mediante el uso de una ventana de alerta (No se enviará el formulario en ese caso).
		AYUDA / SUGERENCIAS:

		Cree una función personalizada asociada al click del botón.
		Apóyese en la herramienta getElementsByTagName para obtener todos los elementos de tipo input 
		(devuelve un vector de elementos, el cual puede ser recorrido con un bucle for).
		
		Para discriminar el tipo de input (los radio o bien el botón), verifique el “type” del elemento.
		
		Apóyese en la propiedad “checked” (devuelve un booleano) de cada uno de los elementos del vector obtenido con 
		getElementsByTagName  para verificar si un radio button está tildado o no.
